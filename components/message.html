Message 消息提示 ​

Message 组件用于展示短暂的消息提示，支持自定义类型、持续时间、偏移量、关闭按钮等功能。

基础用法 ​

通过 message 属性传递消息内容，type 属性设置提示类型，duration 属性设置显示的持续时间。

<script setup>
import { CreateMessage } from '@/components/Message/method'
import Button from '@/components/Button/Button.vue'
import Icon from '@/components/Icon/Icon.vue'
import { ref, h } from 'vue'

const test = ref()
const message = '这是一个简单的消息！'
const messageVNode = h('div', { class: 'custom-message' }, [
  h('strong', '这是一条复杂的消息！'),
  h(Icon, { icon: 'info-circle' }),
])
const open = () => {
  CreateMessage({ message: message, type: 'success' })
}
const open2 = () => {
  CreateMessage({ message: messageVNode, type: 'success' })
}
</script>

<template>
  <Button @click="open">showMessage</Button>
  <Button @click="open2">showVNode</Button>
</template>

<style scoped>
/* 自定义样式 */
</style>

带关闭按钮 ​

可以设置 showClose 为 true，使得消息提示框显示关闭按钮。

<script setup>
import { CreateMessage } from '@/components/Message/method'
import Button from '@/components/Button/Button.vue'
import { ref } from 'vue'

const test = ref()
const message = "这个提示有关闭按钮！"
const open = () => {
  CreateMessage({ message: message,type:'danger', showClose: true,duration:0 })
 
}

</script>

<template>
  <Button @click="open">showMessage</Button>
</template>

<style scoped>
/* 自定义样式 */
</style>

鼠标悬停暂停计时 ​

当鼠标悬停在消息提示上时，自动关闭计时会暂停，移开鼠标后重新开始计时。

<script setup>
import { CreateMessage } from '@/components/Message/method'
import Button from '@/components/Button/Button.vue'
import { ref } from 'vue'

const test = ref()
const message = '鼠标悬停时会暂停自动关闭计时'
const open = () => {
  CreateMessage({ message: message, type: 'success' })
  
}

</script>

<template>
  <Button @click="open">showMessage</Button>
</template>

<style scoped>
/* 自定义样式 */
</style>

Message Attributes ​

属性名说明类型默认值
message消息内容string—
type消息类型enum - 'info' | 'success' | 'warning' | 'danger' | 'info'
duration消息显示时长（毫秒） -如果是0则不会主动消失number3000
offset消息与页面顶部的偏移量number20
transitionName消息出现与消失的过渡名称string'fade-up'
showClose是否显示关闭按钮booleanfalse
